## 一、项目背景

《丑闻制造者归来》（스캔들 메이커가 돌아왔다）是一部韩国网络漫画，讲述女主角 Harriet Listerwell 被表姐 Bella 陷害后，从受害者蜕变为主动复仇的"丑闻制造者"的故事。

本项目旨在为该作品打造一个**粉丝专题网站**，满足读者对作品内容的深度探索需求，同时提供独特的互动体验，增强粉丝社区黏性。

---

## 二、目标用户

| 用户类型 | 描述 |
|----------|------|
| 核心粉丝 | 已读漫画，寻求深度内容（角色分析、剧情梳理） |
| 新读者 | 尚未读漫画，被推荐或搜索而来，需要入门引导 |
| 同人创作者 | 有创作欲望，寻找素材灵感和发布平台 |

## 三、MVP 范围

### 3.1 MVP 包含功能（Phase 1）

#### 🏠 首页
- 全屏暗色背景 + 大标题逐词浮现动效（Framer Motion）
- 副标题文案：*"She was framed. She was broken. Now she is back."*
- 两个 CTA 按钮：进入故事 / 查看角色
- 底部随机展示一条角色经典台词

#### 📖 故事介绍页（/story）
- Harriet vs Bella 人物对比卡片
- 故事时间线（从被陷害 → 觉醒 → 复仇），按章节展开
- 事件类型标签：BETRAYAL / AWAKENING / REVENGE / REVEAL

#### 👥 角色图鉴页（/characters）
- 所有主要角色卡片网格展示
- 每张卡片含：阵营标签、姓名、头衔、描述、特征标签、经典台词
- 阵营区分：主角 / 反派 / 中立 / 盟友

#### 🗂️ 丑闻档案馆（/archives）
- 以"机密文件"形式展示剧情关键事件
- 每份档案含：案件编号、密级印章（TOP SECRET / CONFIDENTIAL）、摘要、可展开详情
- 状态标注：OPEN / CLOSED / ONGOING

#### 🧭 全局导航
- 固定顶部导航栏，含移动端汉堡菜单
- 当前页高亮

---

### 3.2 MVP 不包含（Phase 2 规划）

| 功能 | 说明 |
|------|------|
| 丑闻生成器 | 用户输入名字，生成复古报纸头条图片，可分享 |
| 粉丝评论 / 留言板 | 需引入后端存储，MVP 暂不做 |
| 章节追更提醒 | 邮件订阅，需邮件服务支持 |
| 角色关系图谱 | 可交互关系网，开发成本较高 |
| 同人投稿社区 | 需要用户系统，Phase 3 考虑 |

## 四、技术方案

| 层级 | 选型 | 说明 |
|------|------|------|
| 框架 | Next.js 14（App Router） | 前后端一体，SEO 友好 |
| 样式 | Tailwind CSS + Framer Motion | 快速开发 + 动效支持 |
| 数据 | 静态 TypeScript 文件 | 无数据库，零成本 |
| 图片 | 内存内联 / 静态资源 | 不依赖外部 CDN |
| 部署 | Cloudflare Pages | 边缘网络，全球访问快，免费额度够 |
| 版本管理 | GitHub | 代码托管 + CI/CD 触发部署 |
